# DIALOGFLOW

- DIALOGFLOW IS AN NLP PLATFORM THAT CAN BE USED TO BUILD 
CONVERSATIONAL APPLICATION

- IT IS POWERED BY GOOGLE







# INTEGRATING A DIALOGFLOW CHATBOT INTO A WEBSITE

- CREATE A NEW AGENT WITH
- TRAIN THE CHATBOT BY ADDING THE INTENTS
- CHECK THE FUNCTIONALITY OF THE CHATBOT
- GO TO THE INTEGRATION TAB
- CLICK FOR WEB DEMO AND OBTAIN THE FRAME SCRIPT
- INSERT THE FRAME SCRIPT INTO THE WEB PAGE
