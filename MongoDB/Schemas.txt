=== DESIGNING SCHEMAS FOR MONGODB COLLECTIONS

- MONGOOSE IS USED TO DESIGN THE SCHEMAS FOR THE COLLECTIONS

=== DEFINING A SCHEMA
const userSchema = new Mongoose.Schema({
  username: {
    type: String,
    required: true,
  },
  email: {
    type: String,
  },
  password: {
    type: String,
  },
})

=== DEFINING A MODEL
Mongoose.model("User", userSchema)
